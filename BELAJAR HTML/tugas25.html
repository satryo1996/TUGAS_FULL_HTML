Mengenal Seantic Elemen Di HTML

  • article

    • Ketika kita ingin membuat untuk menampilkan blockchain artikel-artikel yang ada di website kita, tag artikel bisa digunakan element header dan footer. karena tag artikel memiliki type block, artinya element ini memiliki lebar sesuai lebar browser jika kita tidak menentukan lebarnya

  • Aside

    • Element isi bisa kita gunakan untuk melengkapi layot utama. Misal kita memiliki halaman blog maka kita bisa menambahkan aside di bagian bawah postingan untuk menjelaskan identitas dari penulisan article.karena aside memiliki type/sifat block maka lebarnya sesuai page jika kita tidak menentukan lebarnya.
